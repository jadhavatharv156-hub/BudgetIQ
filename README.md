# 💰 BudgetIQ – Personal Budget Management System

BudgetIQ is a web-based personal finance management application developed using **Python, Flask, and SQLite**. The application helps users track their income and expenses, manage their finances efficiently, and generate financial reports.

---

## 📌 Project Overview

Managing personal finances can be difficult when expenses and income are not properly tracked. BudgetIQ provides a simple and user-friendly platform where users can:

* Register and log in securely.
* Add and manage expenses.
* Monitor financial activities.
* View reports and spending patterns.
* Analyze monthly and yearly expenses.

The project aims to make personal budgeting easier and more organized.

## ⚙️ Installation Guide

### 1. Clone the Repository

```bash
git clone https://github.com/jadhavatharv156-hub/BudgetIQ.git
cd BudgetIQ
```

### 2. Create Virtual Environment

```bash
python -m venv .venv
```

### 3. Activate Virtual Environment

**Windows**

```bash
.venv\Scripts\activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the Application

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
